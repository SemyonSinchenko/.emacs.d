* System Dependencies

This configuration relies on the following external CLI and system packages.

** Core Utilities
- =git=: Version control (required by =llm-diff=, =aider=, and =magit=).
- =ripgrep=: Fast text search (used by =consult-ripgrep= and project search).
- =fd=: Fast file search (standard companion to ripgrep).
- =curl=: HTTP client (required for API requests via =gptel=).
- =sqlite3=: Database engine (required by =org-roam=).
- =unzip=: Archive extractor (required by the =reader= package for EPUB files).

** AI Tools
- =aider=: AI pair programming tool.
  - Install via pip: =pip install aider-chat=
  - Requires =git= to be initialized in the project directory.

** Language Servers & Tools

*** Python
- =python3=: Runtime.
- =ty=: LSP server (configured as =("ty" "server")= in =lang-prog.el=).
- =ruff=: Linter and formatter (used by =apheleia= and =flycheck=).

*** Java
- =jdtls=: Eclipse JDT Language Server (required for Java support).

*** Scala
- =metals=: Scala Language Server.
- =coursier=: (Optional but recommended) Installer for Metals.

*** Rust
- =rust-analyzer=: LSP server for Rust.
- =cargo=: Build tool (usually installed with Rust).

*** Markdown
- =multimarkdown=: Processor for Markdown files (configured as =markdown-command=).
